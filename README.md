## 環境構築
下記のコマンドを実行してください

```
cd
mkdir form_with_practice && cd form_with_practice
git clone git@github.com:kenchasonakai/form_with_sample.git
cd form_with_sample
docker compose build
docker compose up
```

サーバーが立ち上がったら下記のURLにアクセスしてください

http://localhost:3001
